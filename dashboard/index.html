üìù Commit Standards

Write commits that generate great announcements

üí°
Better commits = Better announcements

WishlistOps uses your commit messages to generate Steam announcements. Well-structured commits create more engaging content!

‚úÖ Player-Facing Commits

These will trigger announcements:

feat: New features
feat: add grappling hook mechanic
fix: Bug fixes
fix: resolve boss invincibility glitch
perf: Performance
perf: optimize forest level for 60fps

‚è≠Ô∏è Internal Commits

These won't trigger announcements:

chore: Maintenance
chore: update dependencies
docs: Documentation
docs: fix README typo
refactor: Code changes
refactor: extract enemy AI class

üì∏ Screenshot Directive

Tell WishlistOps which screenshot to use with your commit:

feat: add dragon boss fight [shot: screenshots/dragon_boss.png]

Screenshot Priority

  1. Explicit [shot: path] directive
  2. Images changed in the commit
  3. Images in screenshots/, promo/, marketing/

Recommended Folders

  • screenshots/ - gameplay captures
  • promo/ - promotional images
  • marketing/ - marketing assets
